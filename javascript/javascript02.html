01. 변수 : 데이터 불러오기

변수에 데이터를 불러오는 것은 프로그램에서 외부 소스에서 값을 가져와 해당 값을 변수에 할당하는 과정을 의미합니다. 

{
    let x = 100, y = 200, z = "javascript";

    console.log(x, y, z);
}
결과 확인하기
100 200 javascript

02. 상수 : 데이터 불러오기

상수에 데이터를 불러오는 것은 프로그램에서 외부 소스에서 값을 가져와 해당 값을 상수에 할당하는 과정을 의미합니다. 상수는 한 번 할당되면 변경되지 않으므로, 한 번 할당된 값은 프로그램 실행 중에 변경될 수 없습니다. 

{
    const x = 100, y = 200, z = "javascript";

    console.log(x, y, z);
}
결과 확인하기
100 200 javascript

03. 배열 : 데이터 불러오기

배열에 데이터를 불러오는 것은 프로그램에서 여러 값을 하나의 변수에 저장하는 과정을 의미합니다. 이를 통해 프로그램은 하나의 변수명으로 여러 데이터를 효율적으로 관리하고 접근할 수 있습니다. 

{
    const arr = [100, 200, "javascript"];

    console.log(arr[0], arr[1], arr[2]);
}
결과 확인하기
100 200 javascript

04. 배열 : 데이터 불러오기 : 갯수

배열의 "length" 속성은 해당 배열에 저장된 요소의 개수를 나타내며, 이를 통해 프로그램은 배열의 크기를 파악하고 처리할 수 있습니다. 

{
    const arr = [100, 200, "javascript"];

    console.log(arr.length);
}
결과 확인하기
3

05. 배열 : 데이터 불러오기 : for()문

for()문은 프로그래밍에서 반복적인 작업을 수행하기 위해 사용되는 제어 구조 중 하나입니다. 배열과 함께 사용될 때, for()문은 배열의 각 요소에 접근하여 작업을 반복적으로 수행하는 데 유용합니다. 

{
    const arr = [100, 200, 300, 400, 500, 600, 700, 800, 900];

    for (let i = 0; i < arr.length; i++) {
        console.log(arr[i]);
    }
}

06. 배열 : 데이터 불러오기 : forEach()

forEach() 메서드는 배열의 각 요소에 대해 주어진 함수를 실행하는 배열 메서드입니다. 이 메서드를 사용하면 배열의 모든 요소를 반복하면서 원하는 작업을 수행할 수 있습니다. 일반적으로 반복적인 작업을 수행할 때 for 문보다 간편하게 사용할 수 있습니다. 

{
    const arr = [100, 200, 300, 400, 500];

    // 01
    arr.forEach(function (element) {
        console.log(element);           //100~500
    });
    // 02
    arr.forEach(function (element, index) {
        console.log(index);
    });
    // 03
    arr.forEach(function (element, index, array) {
        console.log(array);
    });

    // 화살표 함수
    // 01
    arr.forEach((e) => {
        console.log(e);
    });
    // 02
    arr.forEach(e => {
        console.log(e);
    });
    // 03
    arr.forEach((e) => console.log(e));
}

07. 배열 : 데이터 불러오기 : for of

for of 구문은 반복 가능한 객체의 각 요소에 대해 반복하는 반복문입니다. 배열이나 문자열과 같은 반복 가능한(iterable) 객체(예: 배열, 문자열, 맵, 세트 등)에서 각 요소에 접근하여 작업을 수행할 수 있습니다. 

{
    const arr = [100, 200, 300, 400, 500, 600, 700, 800, 900];

    for (let element of arr) {
        console.log(element);
    }
}

08. 배열 : 데이터 불러오기 : for in

for in 구문은 객체의 열거 가능한 속성을 반복하는 데 사용되는 반복문입니다. 배열의 인덱스나 객체의 속성명과 같은 키에 대해 반복하며, 주로 객체를 순회할 때 사용됩니다. 그러나 배열에 사용할 경우, 배열의 인덱스를 반환하므로 주의해야 합니다. 

{
    const arr = [100, 200, 300, 400, 500, 600, 700, 800, 900];

    for (let element in arr) {
        console.log(arr[element]);
    }
}

09. 배열 : 데이터 불러오기 : map()

map() 메서드는 배열의 각 요소에 대해 주어진 함수를 호출한 결과를 모아 새로운 배열을 반환하는 배열 메서드입니다. 기존 배열을 변경하지 않고 각 요소에 변형을 가할 때 주로 사용됩니다. 예를 들어, 각 요소를 제곱하거나 문자열을 대문자로 변환하는 등의 작업을 수행할 수 있습니다. 

{
    const arr = [100, 200, 300, 400, 500];

    //01
    arr.map(function (element) {
        console.log(element);       //[100~500]         
    });

    arr.map(function (element, index) {
        console.log(index);
    });

    arr.map(function (element, index, array) {
        console.log(array);
    });
}
